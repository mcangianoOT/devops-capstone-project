# devops-capstone-project

![Build Status](https://github.com/mcangianoOT/devops-capstone-project/actions/workflows/ci-build.yaml/badge.svg)


A DevOps Capstone Project is a comprehensive, hands-on assignment that serves as the final, culminating milestone in a DevOps training program or certification. It requires to build and automate an end-to-end software delivery pipeline, taking a raw application and moving it safely into a production cloud environment.
